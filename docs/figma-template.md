# Figma Template Guide (Interactive AI Resume Builder)

Use this guide to recreate the resume template in Figma with sample data.

## Frames

- Desktop: `1440 x 1024`
- Mobile: `390 x 844`

## Grid

- Desktop: 12 columns, 72px margins, 24px gutter
- Mobile: 4 columns, 20px margins, 16px gutter

## Typography

- Heading: Manrope 24 / 32, 600
- Section Title: Manrope 11 / 16, 700, letter spacing 0.2em, uppercase
- Body: Manrope 13 / 20, 400
- Accent: Manrope 12 / 18, 500

## Color Tokens

- Primary: `#0F766E` (teal-700)
- Text: `#0F172A` (slate-900)
- Muted: `#64748B` (slate-500)
- Surface: `#FFFFFF`
- Border: `#E2E8F0` (slate-200)

## Layout

- Header: Name + contact line, bottom border 2px in primary.
- Sections stacked vertically with 24px spacing.
- Skills displayed as chips with 12px radius.
